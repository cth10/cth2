Nobara é melhor que Ruimdowns 11 para gaming (e para codar)

✍️ Celso Takeshi Hamasaki

se vc eh como eu, vc provavelmente ta cansado do windows 11. tipo, vc so quer jogar, trabalhar, estudar, e ver seus animes, né?

mas o windows parece q tenta ser chato.

ele te força a usar uma conta outlook, ele fica te "espiando" o tempo todo (a tal da "telemetria"), ele bota anuncio no menu iniciar (!!!) e ainda eh o lugar favorito dos virus.

mas tem um distro do linux chamado Nobara que eh perfeito pra isso. E tbm eh amigável pra iniciantes no mundo linux.

O "Lado Chato" do Windows 11

antes de falar do nobara, vamos lembrar pq o windows 11 eh tao... eca.

  • Conta Outlook Obrigatoria: pra começar a usar o pc! vc eh forçado a se logar numa conta online. claro, tem como dar bypass, mas o usuario normal nao vai saber fazer isso sem ter que perder tempo pesquisando.
  • Telemetria e Coleta de Dados: o windows 11 eh mto fofoqueiro. ele fica mandando um monte de dados de "diagnostico" (o q vc faz, o q vc abre) pra microsoft o tempo todo. e eh super dificil de desligar 100%.
  • Anuncios Invasivos: ja viu propaganda de app no seu proprio menu iniciar? ou "sugestoes" no seu explorador de arquivos? q horror!
  • Bloatware: ele vem com um monte de app q ninguem pediu (oi, cortana? oi, candy crush? onedrive? microsoft teams? outlook? clipchamp? microsoft edge? etc etc etc)
  • Virus e Segurança: windows é muito mais vulneravel a virus que linux. vc precisa viver com um antivirus rodando e gastando sua cpu o tempo todo, senao vc ta em risco.
  • É caro: ja viu o preço de uma licença original do windows 11?

Enquanto isso, no Nobara!

ai vc instala o nobara. (q eh um fedora linux "arrumado" pra jogos pelo glorious eggroll). e a diferença eh... tudo.

o nobara nao quer seus dados. ele nao te bota anuncio. ele so quer funcionar.

"Out-of-the-Box" (So funciona!)

o maior problema do linux "normal" (tipo o fedora puro) eh q vc tem q configurar um monte de coisa chata. o nobara ja faz td isso pra vc:

  • Codecs de Anime: vc quer ver um .mkv ou o crunchyroll? no nobara, todos os codecs (h.264, h.265) ja vem prontos!
  • Drivers de Controle: sabe a raiva de parear um controle de xbox no bluetooth? no nobara, os drivers xpadneo e xone ja vem no kernel. eh so ligar e ele funciona 100%. provavelmente.
  • Drivers da Nvidia: no nobara, o instalador pergunta se vc tem nvidia e ja instala tudo sozinho. sem terminal.

Feito pra Jogos (Steam ou nao!)

o nobara foi feito pelo mesmo cara q faz o Proton-GE! ele sabe o q faz!

  • Steam: ja vem instalada e pronta.
  • Lutris (pra jogos do DL-Site, GOG, etc): o nobara eh o melhor sistema pra isso. ele ja vem com o lutris e TODAS as dependencias de wine q os jogos de windows precisam. vc nao vai ter aquele erro chato de "falta .net" ou "falta wmp". vc instala o jogo do dl-site (.exe) e ele funciona.

Leve e Rapido!

o nobara nao tem telemetria, nao tem antivirus rodando sem parar, nao tem servico de propaganda. "parado", ele usa metade da ram q o windows 11 usa.

o q isso quer dizer? sobra mais ram e mais cpu pro seu jogo! muitas vezes, os jogos rodam ate mais rapido no nobara do q no windows.

Mas... tem um porem, ne?

sim, tem. o nobara (e o linux) nao eh perfeito.

jogos q usam anti cheats mto agressivos, chamados de anti cheats de nivel kernel (q precisam entrar no coraçao do sistema) n vao rodar.

a lista eh triste:

  • Valorant e LOL (culpa do vanguard)
  • Genshin Impact
  • Destiny 2
  • Call of Duty (os novos)

esses jogos olham pro nobara, nao entendem o sistema, acham q eh um "hack" e fecham. pra esses, infelizmente, vc ainda vai precisar do windows. ou tentar jogar por cloud gaming.

E pra codar?

se pra jogos a diferença era grande, pra coding a diferença eh gigante!

pra umas 90% das linguagens de programaçao, o linux eh mil vezes melhor q o windows!

O Nobara "Fala a Língua" dos Servidores

pensa assim: quando vc faz um site, um app, ou um bot... onde ele vai rodar "na vida real"? num servidor! (la na nuvem, tipo amazon, google, etc.)

e 99,9% desses servidores no mundo inteiro rodam... LINUX!

  • programando no nobara: vc ta programando no mesmo tipo de sistema q o seu codigo vai viver. os comandos sao os mesmos, as pastas (/home/user/) sao iguais, as permissoes sao iguais. se funciona no seu pc, provavelmente funciona no servidor.
  • programando no windows: vc ta num ambiente totalmente diferente (C:\Users\). vc testa no seu pc, ai bota no servidor e... "quebrou!". pq la no servidor as coisas funcionam de um jeito diferente.

Instalando Ferramentas (dnf vs "procurar .exe")

no nobara (fedora), pra instalar qualquer coisa de coding, vc so faz:

sudo dnf install python3
sudo dnf install nodejs
sudo dnf install git

eh rapido, eh seguro (vem do repositorio oficial), e ta tudo num lugar so.

no windows... vc tem q ir no google, achar o site certo (cuidado pra nao baixar virus!), baixar o .exe, clicar em "next, next, next..." e torcer pra ele nao instalar 3 toolbars no seu browser.

A "Gambiarra" do Windows (o WSL)

a propria microsoft sabe q o windows eh tao ruim pra programar, q eles literalmente criaram um "linux dentro do windows"! (chama WSL).

no nobara, vc nao precisa de WSL. vc JA esta no linux! eh o "original", eh mais leve e mais rapido!

Quando o Windows eh melhor?

o windows so ganha se vc for programar coisas especificas da microsoft:

  • apps .net antigos (apps de desktop pra windows).
  • jogos em unity/unreal (se vc quiser "buildar" pra xbox ou windows especificamente).
